# Migdal
### Home work in JAVA with 3 parts:
* Part 1: Server-side development (JAVA) - part_1 folder.
* Part 2: Algorithm - part_2 file.
* Part 3: Design abilities - part_3 file.

## Installation

```bash
git clone https://github.com/Pnina-Klain/Migdal.git
cd project-name
```
## Compile and Run
### Via IDE:
1. Click the Run button or select Run → Run Main
2. Select Main from the Part_1 package
### Via Terminal:
```bash
cd src
javac Part_1/Main.java
java Part_1.Main
```
### Dependencies

- **GSON 2.10.1** (Google) - JSON serialization/deserialization
  - [GitHub](https://github.com/google/gson)
  - Added to project classpath via IDE

#### How to Add GSON to Your IDE

##### IntelliJ IDEA
1. Project Structure → Libraries
2. `+` → Download Library
##### VS code - Manual installation
1. Download GSON from here https://repo1.maven.org/maven2/com/google/code/gson/gson/2.10.1/
2. Add the JAR to lib/ folder
3. Add to classpath on run:
```bash
java -cp "lib/gson-2.10.1.jar:target/classes" Part_1.Main
```
